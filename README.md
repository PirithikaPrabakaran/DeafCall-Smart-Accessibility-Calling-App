DeafCall is an accessibility project that helps hearing impaired users by transcribing live speech on phone calls into text. 
The project consists of an Android application and a web application. 
The Android app is developed in Java and XML. 
It can identify incoming and outgoing calls, initiate a background service and change speech to text using the device microphone. 
Features include real-time transcription, call duration tracking and mute/unmute controls. 
The web app uses HTML, CSS, and JavaScript and mimics a call interface, using the Web Speech API to convert speech into live subtitles.
It also offers features such as adjustable font size, conversation history, and a user-friendly interface to improve accessibility.

The main objective of this project is to improve the communication of deaf and hard-of-hearing people through a real-time transcription system. 
However, the system uses microphone input, not the call audio itself, and may be less accurate depending on the background noise and quality of the device. 
Future enhancements may include multi-language support, cloud-based transcript storage, real call integration, and AI-based improvements for enhanced accuracy and performance.
